JAAT-AI UI Components

A collection of UI components built with plain HTML, CSS, and JavaScript for JAAT-AI applications.

Buttons Dialogs Select Dropdowns Usage Guide

Button Component

Versatile button components with various styles, states, and icon options.

<button class="jaat-button">Primary Button</button>

Dialog Component

Modal dialogs for displaying information, alerts, or forms.

Demo Dialog

This is a sample dialog component.

<button class="jaat-button" onclick="jaatUI.dialog.open('myDialog')">Open Dialog</button>

Select Component

Dropdown selects for choosing from a list of options.

<div class="jaat-select"> <label for="mySelect" class="jaat-select-label">Select Label</label> <div class="jaat-select-wrapper"> <select id="mySelect" class="jaat-select-input"> <option value="option1">Option 1</option> </select> <div class="jaat-select-icon">...</div> </div> </div>

Usage Guide

How to use these components in your JAAT-AI project.

  1. Include the component CSS files in your HTML head section: 
    <link rel="stylesheet" href="path/to/button.css">
  2. Include the component JavaScript files before the closing body tag: 
    <script src="path/to/button.js"></script>
  3. Use the component HTML structure in your templates
  4. Access component APIs via the global jaatUI object
